# 朝活情報アプリ

まずは書き込みと読み込みをしっかりやろうということで、  
機能も書き込みと読み込みというようにシンプルになってます


## 工夫した点
* 入力してもらった内容の一部だけ表示したかったので、CSVファイルの内容を key value の形で受け取った事
* 画面は他のサイトを参考にしました!!
* フォームの入力が世界で一番嫌いなんですが、テストデータを入れるのが大変だったのでぐるなびのAPIからランダムで入力できるようにしました
